# Literals

## Numerical literals
```cpone
7     // i32 literal
9u    // u32 literal
1i64  // i64 literal
1u64  // u64 literal
0o644 // An octal literal
0x7af // A hexadecimal literal
7.0f  // f32 literal
95.0  // f64 literal
```

## Character literals
```cpone
''a   // 'a' character
''7   // Number '7'
''\n  // New-line character
''\\  // Slash character
''"   // Space character
```

## String literals
```cpone
"hello\nworld"        // hello(new line)World
'"hello\nworld"       // hello\nworld
'<hello\nworld>       // hello\nworld
'[hello\nworld]       // hello\nworld
'{hello\nworld}       // hello\nworld
'(hello\nworld)       // hello\nworld

// hello\nworld}">]) // even comments
'- hello\nworld}">]) // even comments

// hello\nworld}">]) // even comments(new line)
'= hello\nworld}">[) // even comments
```
